Scene 4 · Spin the wheel

A skin over the same engine — the server spins, the wheel performs

Scene 5 · The organiser's wallet

Promo pools — budgeted, targeted, burn-down visible

Codes are generated in Howler ticketing and uploaded into pools with rules (min quantity, ticket types, expiry). Pink chips target history tiers; purple chips target signals. Issuance stops dead when a pool empties. Ships on both surfaces: Admin → client → Loyalty, and the client's own Settings.
