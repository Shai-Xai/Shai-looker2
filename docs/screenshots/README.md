# Screenshots for the Value Case

Drop product screenshots here using the **exact filenames** below. The gallery in
`docs/value-case.html` ("See it in action") loads them automatically — until a
file exists, that tile shows a labelled placeholder, so the page never breaks.

| Filename | What to capture |
|---|---|
| `01-onboarding.png` | Client onboarding flow — spinning up a new client |
| `02-branded-dashboard.png` | A white-labelled, per-client dashboard |
| `03-engagement-engine.png` | The engagement engine / abandoned-cart campaign builder |
| `04-segments-audiences.png` | Segment builder + Meta/TikTok audience sync |
| `05-insight-digest.png` | The insight digest / daily email with suggested actions |
| `06-settlements.png` | Settlements / invoice send with client notification |

**Tips**
- Use a **16:10** crop (the frame is `aspect-ratio:16/10`, top-aligned).
- Aim for ~1600px wide PNGs so they stay crisp on retina screens and in PDF export.
- Keep any sensitive client data out of the shots (use demo/sample data).
